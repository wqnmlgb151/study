## 笔记约定

- 文件与目录使用中文命名
- 使用标准 Markdown 相对链接
- 附件图片存放于各目录下的 `assets/` 子目录
- 代码块标注语言类型以便语法高亮
- 提交信息格式：`vault backup: YYYY-MM-DD HH:mm:ss`（由 obsidian-git 自动生成）
