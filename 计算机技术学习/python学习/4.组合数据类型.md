## 1.序列和索引
### 序列
- **序列**是一个用于存储连续多个值的连续空间，每个值都对应一个整数的编号，称为**索引**
- 序列类型包括字符串、列表、元组、集合和字典
### 索引
- 正向**递增**索引 从左到右 从0开始
- 反向**递减**索引 从右到左 从-1开始
```python
s = 'helloworld'
for i in range(0,len(s)):
	print(i,s[i],end='\t\t')
for i in range(-10,0):
	print(i,s[i],end='\t\t')
```
### 切片操作的语法结构
> 切片操作可用于所有序列类型，最后生成新的序列

`序列[start:end:step]` 
- `start`切片开始的索引（包含）
- `end`切片结束的索引（不包含）
- `step`步长（默认为1）
```python
s = 'helloworld'
sl = s[0:5:2]
print(sl)#hlo
#省略开始位置，默认从0开始
print(s[:5:2])
print(s[:5:])
#省略结束位置默认到结尾（包含最后一个元素）
print(s[0::1])

print(s[5::])
print(s[5:])#同上
#步长为负数
print(s[::-1])#将s逆序输出
print(s[-1:-11:-1])#同上

```
### 序列相关操作
#### 序列相加
```python
s1 = 'hello'
s2 = 'world'
print(s1+s2)
```
#### 序列相乘
```python
s = 'h'
print(s*5)
print('-'*40)
```
#### 序列操作函数或操作符
- `x in s`如果x是s的元素，结果为True，否则为False
- `x not in s`与上相反
- `len(s)`序列s中的元素个数即s序列长度
- `max(s)`序列s中元素最大值
- `min(s)`序列s中元素最小值
- `s.index(x)`序列s中第一次出现x的位置
- `s.count(x)`序列s中x出现的总次数
```python
s = 'hello'
print('e' in s)
print('v' in s)

print('e' not in s)
print('v' not in s)


```
