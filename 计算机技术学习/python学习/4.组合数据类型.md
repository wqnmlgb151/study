## 1.序列和索引
### 序列
- **序列**是一个用于存储连续多个值的连续空间，每个值都对应一个整数的编号，称为**索引**
- 序列类型包括字符串、列表、元组、集合和字典
### 索引
- 正向**递增**索引 从左到右 从0开始
- 反向**递减**索引 从右到左 从-1开始
```python
s = 'helloworld'
for i in range(0,len(s)):
	print(i,s[i],end='\t\t')
for i in range(-10,0):
	print(i,s[i],end='\t\t')
```
### 切片操作的语法结构
> 切片操作可用于所有序列类型，最后生成新的序列

`序列[start:end:step]` 
- `start`切片开始的索引（包含）
- `end`切片结束的索引（不包含）
- `step`步长（默认为1）
```python
s = 'helloworld'
sl = s[0:5:2]
print(sl)#hlo
#省略开始位置，默认从0开始
print(s[:5:2])
print(s[:5:])
#省略结束位置默认到结尾（包含最后一个元素）
print(s[0::1])

print(s[5::])
print(s[5:])#同上
#步长为负数
print(s[::-1])#将s逆序输出
print(s[-1:-11:-1])#同上

```
### 序列相关操作
#### 序列相加
```python
s1 = 'hello'
s2 = 'world'
print(s1+s2)
```
#### 序列相乘
```python
s = 'h'
print(s*5)
print('-'*40)
```
#### 序列操作函数或操作符
- `x in s`如果x是s的元素，结果为True，否则为False
- `x not in s`与上相反
- `len(s)`序列s中的元素个数即s序列长度
- `max(s)`序列s中元素最大值 返回该元素
- `min(s)`序列s中元素最小值 返回该元素
- `s.index(x)`序列s中第一次出现x的位置
- `s.count(x)`序列s中x出现的总次数
```python
s = 'hello'
print('e' in s)
print('v' in s)

print('e' not in s)
print('v' not in s)

print(len(s))
print(max(s))
print(min(s))
print(s.index('e'))
print(s.count('l'))
```
## 2.列表
- 一系列按特殊排列顺序的元素组成
- 是python中内置的**可变序列**
- 在python中使用`[]`定义列表，元素与元素之间使用**英文逗号**分隔
- 列表中的元素可以是**任意**的**数据类型**
- 对序列的操作符、运算符、函数均可使用
### 列表的创建方法
1. 使用`[]`直接创建
2. 使用内置函数`list()`创建列表
```python 
列表名=[element1,element2........elementN]
列表名=list(序列)
```
### 列表的删除
```python

```

### 列表遍历
1. `for`循环
2. `for`循环+索引
3. `enumerate()`函数
	- 将可迭代对象（列表、元组、字符串等）转化为一个索引-元素对的迭代器
	- `enumerate(lst,start=0)`表示转化lst，从索引0号位开始，后面的start可省略，默认从0开始
	- start=1即表示将新输出时元素索引下标1开始但是元素的输出和顺序不变，只是索引起始值变了
```python
lst = ['hello','php','world','python']

for item in lst:
	print(item)

for i in range(0,len(lst)):
	print(lst[i])

for index,item in enumerate(lst):#这里的index和item是自己定义的变量，分别用于接收索引与元素
	print(index,item)
for index,item in enumerate(lst,start=1):
	print(index,item)
for index,item in enumerate(lst,1):
	print(index,item)
```
### 列表的相关操作
- `lst.append(x)`在lst最后增加一个元素
- `lst.insert(index,x)`在列表索引index处增加元素x
- `lst.claer()`清除lst中的所有元素
- `lst.pop(index)`将lst中索引index出的元素取出并删除
- `lst.remove(x)`将列表lst出现的第一个x元素删除
- `lst.reverse()` 将列表中的元素反转不产生新的元素
- `lst.copy()`拷贝lst中的所有元素并生成一个新的列表
```python
lst = ['hello','world','python']
print(lst)
lst.append('sql')
print(lst)
lst.insert(1,100)
print(lst)
lst.remove('100')
print(lst)
#pop(index)  现根据索引将元素取出再删除
print(lst.pop(1))
print(lst)

lst.reverse()
print(lst)

lst.clear()
print(lst)

new_lst=lst.copy()
print(new_lst)
```
