## 1.序列和索引
### 序列
- **序列**是一个用于存储连续多个值的连续空间，每个值都对应一个整数的编号，称为**索引**
- 序列类型包括字符串、列表、元组、集合和字典
### 索引
- 正向**递增**索引 从左到右 从0开始
- 反向**递减**索引 从右到左 从-1开始
```python
s = 'helloworld'
for i in range(0,len(s)):
	print(i,s[i],end='\t\t')
for i in range(-10,0):
	print(i,s[i],end='\t\t')
```
### 切片操作的语法结构
> 切片操作可用于所有序列类型，最后生成新的序列

`序列[start:end:step]` 
- `start`切片开始的索引（包含）
- `end`切片结束的索引（不包含）
- `step`步长（默认为1）
```python
s = 'helloworld'
sl = s[0:5:2]
print(sl)#hlo
#省略开始位置，默认从0开始
print(s[:5:2])
print(s[:5:])
#省略结束位置默认到结尾（包含最后一个元素）
print(s[0::1])

print(s[5::])
print(s[5:])#同上
#步长为负数
print(s[::-1])#将s逆序输出
print(s[-1:-11:-1])#同上

```
### 序列相关操作
#### 序列相加
```python
s1 = 'hello'
s2 = 'world'
print(s1+s2)
```
#### 序列相乘
```python
s = 'h'
print(s*5)
print('-'*40)
```
#### 序列操作函数或操作符
- `x in s`如果x是s的元素，结果为True，否则为False
- `x not in s`与上相反
- `len(s)`序列s中的元素个数即s序列长度
- `max(s)`序列s中元素最大值 返回该元素
- `min(s)`序列s中元素最小值 返回该元素
- `s.index(x)`序列s中第一次出现x的位置
- `s.count(x)`序列s中x出现的总次数
```python
s = 'hello'
print('e' in s)
print('v' in s)

print('e' not in s)
print('v' not in s)

print(len(s))
print(max(s))
print(min(s))
print(s.index('e'))
print(s.count('l'))
```
## 2.列表
- 一系列按特殊排列顺序的元素组成
- 是python中内置的**可变序列**
- 在python中使用`[]`定义列表，元素与元素之间使用**英文逗号**分隔
- 列表中的元素可以是**任意**的**数据类型**
- 对序列的操作符、运算符、函数均可使用
### 列表的创建方法
1. 使用`[]`直接创建
2. 使用内置函数`list()`创建列表
```python 
列表名=[element1,element2........elementN]
列表名=list(序列)
```
### 列表的删除
```python
del 列表名
```

```python
lst=['hello','world',98,100.5]
print(lst)

lst2 = list('helloworld')#每个元素都是单独的字符
lst3 = list(range(1,10,2))#元素都是奇数

print(lst+lst2+lst3)
print(lst*3)
print(len(lst))
print(max(lst))
print(min(lst))
print(lst2.index('o'))
print(lst2.count('o'))
del lst
```
### 列表遍历
1. `for`循环
2. `for`循环+索引
3. `enumerate()`函数
	- 将可迭代对象（列表、元组、字符串等）转化为一个索引-元素对的迭代器
	- `enumerate(lst,start=0)`表示转化lst，从索引0号位开始，后面的start可省略，默认从0开始
	- start=1即表示将新输出时元素索引下标1开始但是元素的输出和顺序不变，只是索引起始值变了
```python
lst = ['hello','php','world','python']

for item in lst:
	print(item)

for i in range(0,len(lst)):
	print(lst[i])

for index,item in enumerate(lst):#这里的index和item是自己定义的变量，分别用于接收索引与元素
	print(index,item)
for index,item in enumerate(lst,start=1):
	print(index,item)
for index,item in enumerate(lst,1):
	print(index,item)
```
### 列表的相关操作
- `lst.append(x)`在lst最后增加一个元素
- `lst.insert(index,x)`在列表索引index处增加元素x
- `lst.claer()`清除lst中的所有元素
- `lst.pop(index)`将lst中索引index出的元素取出并删除
- `lst.remove(x)`将列表lst出现的第一个x元素删除
- `lst.reverse()` 将列表中的元素反转不产生新的元素
- `lst.copy()`拷贝lst中的所有元素并生成一个新的列表
```python
lst = ['hello','world','python']
print(lst)
lst.append('sql')
print(lst)
lst.insert(1,100)
print(lst)
lst.remove('100')
print(lst)
#pop(index)  现根据索引将元素取出再删除
print(lst.pop(1))
print(lst)

lst.reverse()
print(lst)

lst.clear()
print(lst)

new_lst=lst.copy()
print(new_lst)

#例表元素的修改
#根据索引进行修改
lst[1]='mysql'

```
### 列表排序
#### 1. 列表对性的 sort 方法
`lst.sort(key=None,reverse=False)` 
- `key` 表示排序的规则
- `Reverse` 表示排序方式、默认为升序、默认值为 False
- 返回值为 None 直接对原序列排序，不会产生新的序列
```python
lst=[12,4536,123,5672,34]

#升序
lst.sort()#排序在原列表基础上进行，不会产生新列表
print(lst)

#降序
lst.sort(reverse=True)
print(lst)

lst2=['banana','apple','Cat','Orange']

#升序
lst2.sort()
print(lst2)

#降序
lst2.sort(reverse=True)
print(lst2)

#自己指定规则_忽略大小写
lst2.sort(key=str.lower)#先全部转化为小写，再排序
print(lst2)
```
#### 2. 内置函数 sorted（）
`sorted(iterable,key=None,reverse=False)`
- `iterable` 表示排序对象
- 返回值为序列，原序列保持不变
```python
lst=[12,4536,123,5672,34]
print(lst)

lst1=sorted(lst)
print(lst1)

lst2=sorted(lst,reverse=True)
print(lst2)#降序

lst_1=['banana','apple','Cat','Orange']
print(lst_1)

lst3=sorted(lst_1,key=str.lower)
print(lst3)
```
### 列表生成式
语法结构

```python
print(lst)
```

## 3. 元组
>  python 中内置的不可变序列
>  使用 () 定义元组，元素与元素之间用英文, 分隔
>  元组中只有一个元素是，逗号也不能省略
>  对序列的操作符、运算符、函数大部分均可使用
### 元组的创建
- 直接创建
语法结构：`元组名=(element1,element2,......,elementN)`
- 使用内置函数 tuple () 创建元组
语法结构：`元组名=tuple(序列)`
### 元组的删除
`del 元组名`
```python
t=('hello',[10,20,20],'python','world')
print(t)

t=tuple('helloworld')
print(t)

t=tuple([10,20,30])
print(t)

print(10 in t)
print(10 not in t)

print(max(t))
print(min(t))
print(len(t))
print(t.index(10))
print(t.count(10))

#元组中只有一个元素是，逗号也不能省略
t=(10,)#不加,的话类型为int
print(type(t))

```

### 元组的访问与遍历
1. 可通过索引访问或遍历，类似列表
2. 通过 for 循环进行遍历
3. 使用函数 enumerate () 进行遍历
```python
t=('python','hello','world')
print(t[0])
t1=t[0:3]
print(t1)
#for循环遍历
for item in t:
	print(item)

for i in range(len(t)):

	print(index,item)
for index,item in enumerate(t,start=11):
	print(index,item)#index从11开始剩下不变
	
```
### 元组生成式


## 4. 字典
- 根据一个信息查找另一个信息的方式构成了“键值对” key-value
- 键必须是不可变数据类型，且在字典中唯一，键相同时 value 进行覆盖，以后面的为准
- 字典亦属于序列，部分序列的函数依旧可以使用
- Key 本是是无序的，但是 3.6 版本对解释器进行了处理，是的输出顺序和添加顺序是一致的
### 字典的创建
1. 直接使用{}创建字典
2. 使用内置函数 dict () 创建字典
3. 使用映射函数 zip 创建字典 zip (lst 1, lst 2) lst 1 是 key lst 2 是 value  最终结果是个 zip 对象要再次进行转换 dict (zip)
```python
d={key1:value1,key2:value2,...}

lst1=(10,20,30,40)
lst2=('cat','dog','zoo','pat')
zipdic=zip(lst1,lst2)
d=dict(zipdic)

dict(key1=value1,key2=value2...)


print(max(d))
print(len(d))
print(min(d))
```
### 字典的删除
- 使用内置函数 del 进行删除 del 字典名
```python
del d
```
### 字典的访问与遍历
#### 字典的取值
- 没有索引，于是通过 key 来实现
- `d[key]` 或 `d.get[key]`
- 两者之间亦有区别，当 key 不存在时：
	- `d[key]` 会报错
	- `d.get(key)` 可以指定默认值
#### 字典的遍历
1. 遍历出 key 与 value 的元组
2. 分别遍历出 key 与 value
```python
for element in d.items():
	pass	

for key,value in d.items():
	pass
```

```python
d={'hello':10,'world':20,'python':30}

print(d['hello'])
print(d.get('hello'))
#展示区别
print(d['java'])#报错
print(d.get('java'))#输出None
print(d.get('java','不存在'))#输出 不存在

for item in d.items():
	print(item)#输出由key和value组成的元组

for key,value in d.items():
	print(key,value)

```
### 字典的相关操作 
- `d.keys()` 获取所有 key 数据
- `d.values()` 获取所有 value 数据
- `d.pop(key,default)` key 存在则获取相应的 value 值，并删除原本 key-value 对；否者获取默认值
- `d.popitem()` 从字典中取出最后一个 key-value 对，结果为元组类型，同时将该 key-value 对从字典中删除
- `d.claer()` 清空字典中的所有 key-value 对
```python
d={1001:'李梅',1003:'王华',1003:'张锋'}
print(d)

#向字典添加元素
d[1004]='张丽丽'#直接使用复制运算符添加元素
print(d)

#获取字典中所有的key
dkeys=d.keys()
print(dkeys)#dict_keys([1001,1002,1003,1004])
print(list(dkeys))
print(tuple(dkeys))

#获取字典中的所有value
dvalues=d.values()
print(dvalues)#dicct_values(['李梅','王华','张锋','张丽丽'])
print(list(dvalues))
print(tuple(dvalues))

#将字典中的数据转换成键值对的形式，以元组形式进行展现
lis=list(d.items())
print(lst)

d=dict(lis)
print(d)

#使用pop函数
print(d.pop(1001))
print(d)

print(d.pop(1008,'不存在'))

#使用popitem函数
print(d.popitem())
print(d)

#清空字典数据
d.clear()
print(d)
```
### 字典的生成式
**语法示范：**
`d={key:value for item in range}`
`d={key:value for key,value in zip(lst1,lst2)}`

## 5.集合
>  python 中的集合和数学中的集合概念一样
>  python 中的集合是个**无序的不重复**元素序列, 部分序列的操作方法依旧可以使用
>  集合中只能存储不可变数据类型
>  在 Python 中集合使用 `{}` 定义
>  与列表、字典一样，都是 python 中的可变数据类型
### 集合的创建
1. 直接使用{}创建集合
	1. **语法结构：**`s={element1,element2,...elementN}`
2. 使用内置函数 set () 创建集合
	1. **语法结构**`s=set(可迭代对象)`
### 集合的删除
**语法结构：**
`del 集合名`
```python
s={10,20,30,40}
print(s)

s=set()#创建一个空集合,空集合的bool值是false
print(s)
s={}#创建的是一个空字典，非集合
print(type(s))

s=set("helloworld")
print(s)#重复元素只显示一个

s1=set([10,20,30])
print(s1)

s2=set(range(1,10))
print(s2)

#集合属于序列
print(max(s2))
print(min(s2))
print(len(s2))

print(9 in s2)
print(9 not in s2)

del s2
```
### 集合相关操作（类数学）
- 交集
- 并集
- 差集
- 补集
```python
A={10,20,30,40,50}
B={30,50,88,76,20}

#交集
print(A&B)

#并集
print(A|B)

#差集
print(A-B)

#补集,不同时包含在A B两个集合的元素
print(A^B)
```
### 集合的遍历
- 使用 for 循环直接进行遍历
- 使用 enumerate () 函数进行遍历
```python
s={10,20,30}
for item in s:
	print(item)

for index,item in enumerate(s):
	print(index,item)
```
### 集合的相关操作方法
- `s.add(x)` 如果 x 不在集合 s 里面，则将 x 加入集合 s
- `s.remove(x)` 如果 x 在集合 s 中，则将其删掉，如果不在，程序报错
- `s.clear()` 清除集合中的所有元素
```python
s={10,20,30}
s.add(40)
print(s)

s.remove(40)
print(s)

s.clear()
print(s)#输出set()  表示空集合
```
### 集合生成式
**语法示范：**
`s={item for item in range}`
`s={item for item in range if true}`
```python
s={i for i in range(10)}
print(s)

s={i for i in range(10) if i%2==0}
print(s)
```
## 6.总结与区分

| 数据类型     | 列表类型  | 元素是否可重复               | 是否有序 | 定义符号        |
| -------- | ----- | --------------------- | ---- | ----------- |
| 列表 list  | 可变序列  | 可重复                   | 有序   | `[]`        |
| 元组 tuple | 不可变序列 | 可重复                   | 有序   | （）          |
| 字典 dict  | 可变序列  | Key 不可重复<br>Value 可重复 | 无序   | （key:value） |
| 集合 set   | 可变序列  | 不可重复                  | 无序   | {}          |
