## 3.数据类型
### 整型 int

- 即是整数，包括正整数、负整数和零 不可变数据类型
- 不可变数据类型为
- 进制种类包括：
	- 十进制
	- 二进制
	- 八进制
	- 十六进制
```python
num=987#默认是十进制，表示整数
num2=0b1010101#二进制
num3=0o07012#八进制
num4=0x87adf#十六进制
print(num,,num2,num3,num4)#输出结果以十进制显示
```
### 浮点数类型 float
- 表示带有小数点的数值，由整数和小数部分组成
- 可使用科学计数法表示，e
```python
height = 173.0
print(height)
print(type(height))#type()查看数据类型

x = 10
y = 10.0
print(type(x))#整型
print(type(y))#浮点数类型

x = 1.99E3#表示x=1990.0
print(x)
```
#### 注意事项：
- 两个浮点类型的数进行计算时会有一定概率运算结果后增加一些***不确定***的尾数
- 例子如下：
```python
print(0.1+0.2)
print(round(0.1+0.2,1))#使用round函数将运算0.1+0.2得出的结果保留一位小数
```
### 复数类型
- python中的复数和数学中的复数概念一样，形式完全一样，也有实部和虚部组成
- 实数部分用.real表示，虚数部分由.imag表示    j=(-1)^(1/2)  j^2=-1
- 复数不是浮点数类型，但是其实数部分和虚数部分的本质都是浮点数
```python
x = 12.3+4.5j
print("x的实数部分:",x.real)
print("x的虚数部分:",x.imag)
```
### 字符串类型
- 使用最多的数据类型之一
- 连续字符序列 可以表示计算机所能识别的一切字符 不可变数据类型
- 有序字符序列 对字符串中某个字符的检索称为索引 
	- 从左到右 从0开始递增
	- 从右往左 从-1开始递减
- **字符串的界定符**：单引号、双引号、三引号
- **转义字符**：由`\`和后面相邻的字符组成，有特殊的作用，来表示新含义，例：
	- `\n` 换行符
	- `\t` 水平制表位，用于横向的跳到下一个制表位（8个字符） 即Tab键
	- `\"` 双引号的正常输出
	- `\'` 单引号的正常输出
	- `\\` 反斜杠的正常输出
- **原字符**：使转义字符失效的字符，`r`或`R`
```python
city='新疆维吾尔自治区'

# 三引号用于定义多行字符串

info2="""我是你爹
	你亲爱的大爹
	来吧尽情的赞美我吧"""
print(r"\n")#输出\n而不是换行
```
#### 切片操作
- 对字符串某个子串或区间的检索叫作**切片**
**语法结构**：
`字符或字符串变量[N:M]` 从n开始到m，但是不包含m
```python
s = 'hello world'
print(s[0],s[-11])#输出结果相同
print('北京欢迎你'[4])#输出该字符串中索引为4的
print('北京欢迎你'[-1])#输出结果如上

print(s[2:7])#输出s从2开始7结束，不包括7
print(s[-8:-3])#输出结果如上
```
