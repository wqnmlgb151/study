### 标识符
标识符的命名规则

+ 可以是字符（中、英）、下划线“_”和数字，并且第一个字符数不能是数字
+ 不能使用Python中的保留字
+ 标识符严格区分大小写
+ 以下划线开头的标识符有特殊意义，一般应避免使用相似的标识符
+ 允许中文，但不建议

+ 变量名不能用保留字

## 3.数据类型
### 整型 int

- 即是整数，包括正整数、负整数和零
- 不可变数据类型为
- 进制种类包括：
	- 十进制
	- 二进制
	- 八进制
	- 十六进制
```python
num=987#默认是十进制，表示整数
num2=0b1010101#二进制
num3=0o07012#八进制
num4=0x87adf#十六进制
print(num,,num2,num3,num4)#输出结果以十进制显示
```
### 浮点数类型 float
- 表示带有小数点的数值，由整数和小数部分组成
- 可使用科学计数法表示，e
```python
height = 173.0
print(height)
print(type(height))#type()查看数据类型

x = 10
y = 10.0
print(type(x))#整型
print(type(y))#浮点数类型

x = 1.99E3#表示x=1990.0
print(x)
```
#### 注意事项：
- 两个浮点类型的数进行计算时会有一定概率运算结果后增加一些***不确定***的尾数
- 例子如下：
```python
print(0.1+0.2)
print(round(0.1+0.2,1))#使用round函数将运算0.1+0.2得出的结果保留一位小数
```
