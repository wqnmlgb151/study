## 1.程序的组织结构
### 顺序结构
> 按照程序语句的自然顺序，从上到下，依次执行每条语句的程序
```python
name = '刘'
age = 19
a=b=c=d=1
a,b,c,d='room'#a='r' b='o' c='o' d='m'
print(a,b,c,d)

name = input("请输入你的名字：")
age = eval(input("请输入你的年龄："))
luck_number = eval(input("请输入你的幸运数字："))
print(name,age,luck_number) 
```
### 分支/选择结构
> 按照条件选择执行不同的代码段
#### 单分支结构if的语法结构
```python
if 表达式:
	语句块
```
- 如果表达式结果为True，则执行语句块；为False，则跳过语句块，执行后面部分
- 如果语句块只有一句，可直接跟在:后面
```python
number=eval(input("请输入你的彩票密码（6位）："))
if number == 951746:
	print("恭喜你中大奖了")
n = 98
if not n%2:
	print(n,"是偶数")
x = input("请输入字符串")
if x:
	print(x,"不是空字符串")
if not x:
	print(x,"是空字符串")
```
#### 双分支结构if-else的语法结构
```python
if 表达式:
	语句块1
else:
	语句块2
```
- 如果表达式为True，则执行语句块1；为False，则执行语句块2
```python
number = eval(input("请输入你的中奖号码："))
if number == 951746:
	print("恭喜你中大奖了")
else:
	print("恭喜你，没中奖")
#简化
result = "恭喜你中大奖了" if number == 951746 else "恭喜你，没中奖"
print(result)

print("恭喜你中大奖了" if number == 951746 else "恭喜你，没中奖")
```
#### 多分支结构if-elif-else的语法结构

```python
if 表达式1:
	语句块1
elif 表达式2:
	语句块2
#......
else:
	语句块n+1
```

- 哪个表达式为True，就执行对应的语句块；如果都不满足，则执行else对应的语句块
```python
score = eval(input("请输入成绩："))
if 0<score or soure>100:
	print("你妈炸了")
elif 0<=score<60:
	print('E')
elif 60<=score<70:
	print('D')
elif 70<=score<80:
	print('C')
elif 80<=score<90:
	print('B')
else:
	print('A')
```
#### if语句的嵌套使用
> 单分支结构、双分支结构与多分支结构在实际开发中是可以嵌套使用的
- 内层的分支结构作为外层的分支结构的语句快使用
```python
answer = input("请问你喝酒了吗？")
if answer == 'y':
	proof=eval(input("请输入你血液中的酒精浓度："))
	if proof<20:
		print("不构成酒驾")
	elif 20<=proof<80:
		print("构成酒驾")
	else proof>=80:
		print("已构成醉驾")
else:
	print("请离开")
```
#### 多个条件的连接
- `and`同时满足多个条件才能执行
- `or`满足任意一个条件就能执行
#### 模式匹配match-case（加强版switch）
- python版本应大于等于3.11
- 只执行条件匹配上的语句块，与c语言switch不同之处在于不会在执行完当前语句块后顺序向下执行
```python
score = input("请输入你的成绩等级：")
match score:
	case 'A':
		print("优秀")
	case 'B':
		print("优秀")
	case 'C':
		print("中等")
	case 'D':
		print("及格")
	case 'E':
		print("不及格")
```
### 循环结构
#### for循环遍历语法结构
```python
for 循环对象 in 遍历对象:
	语句块
```
#### for-else语法结构
> 循环正常执行完毕，才执行else对应的语句块
```python
for 循环对象 in 遍历对象:
	语句块1
else:
	语句块2
```
#### range()函数
> python内置函数
- `range(n,m)`产生一个`[n，m）`的整数序列
- `range(m)`产生一个`[n,m)`的整数序列
- `range(n,m,s)`产生一个`[n,m)`间隔为s的整数序列
```python
for i in range(1,11):
	print(i)
	if i%2==0:
		print("是偶数")
for i in "HELLO":
	print(i)	
s = 0
for i in range(11):
	s+=i
print(s)

for i in range(100,1000):
	if (i%10)**3+(i//10%10)**3+(i//100)**3==i:
		print(i)

s = 0
for i in range(1,11):
	s+=i
else:
	print("1到10累加为:")
```
