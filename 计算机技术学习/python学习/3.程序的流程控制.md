> 按照程序语句的自然顺序，从上到下，依次执行每条语句的程序
```python
name = '刘'
age = 19
a=b=c=d=1
a,b,c,d='room'#a='r' b='o' c='o' d='m'

luck_number = eval(input("请输入你的幸运数字："))

```
### 分支/选择结构
> 按照条件选择执行不同的代码段
#### 单分支结构if的语法结构：
```python
if 表达式:
	语句块
```
- 如果表达式结果为True，则执行语句块；为False，则跳过语句块，执行后面部分
- 如果语句块只有一句，可直接跟在:后面
```python
number=eval(input("请输入你的彩票密码（6位）："))
if number == 951746:
	print("恭喜你中大奖了")
n = 98
if not n%2:
	print(n,"是偶数")
x = input("请输入字符串")
if x:
	print(x,"不是空字符串")
if not x:
	print(x,"是空字符串")
```
