## 1.基本输出函数print
### 语法结构：
print(输出内容)

### print函数的完整输出格式：
print(value,......,sep='',end='\n',file=None)

```python
a=100
b=90
print(20)
print(a)
print(a*b)

print('欢迎你！')
print("欢迎你！")
print('''欢迎你！''')
print("""欢迎你！""")
```

```python
a=100
b=90
print(a,b,'要么出众，要么出局')
```

```python
print('b')
print(chr(98))
#chr函数使的数值转换为对应ASCLL表上的值
```

```python
print(ord('北'))
print(ord('京'))
#ord函数将字符转换为对应的Unicode码
print(chr(21271),chr(20140))
```

```python
fp=open('note.txt','w')#打开/创建note.txt 读写权限 在同一目录下
print("北京欢迎你！",file=fp)
fp.close()#打开文件后必须要关闭
```

```python
print('wcnm',end=' ')
print('nmzl')
```

```python
print("北京欢迎你"+'2008')#+只能连接字符串和字符串
```

## 2.基础输入语句input
### 语法结构
x=input('提示文字')

无论输入数据类型是什么，x的数据类型都是字符串

```python
name=input('请输入你的名字：')
print("我的名字是："+name)
```

```python
num=int(input('请输入你的幸运数字：'))#通过内置函数强制将输入数据类型转换为int类型 其他类型同理可得
print(num,type(num))#ttype()表明num的数据类型
```
