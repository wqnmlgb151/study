> 创建变量或常量时必须指出相应的数字类型，否则无法给变量分配内存

## 1.整型
整型变量表示的是整形类型的数据
short  2个字节
int      4个字节
long   windows占用4字节，linux（32位）占用4字节，（64位）占用8字节
long long  8个字节
```C++
#include<iostream>
using namespace std;

int main()
{
	short num1=10;
	int num2=100;
	long num3=1000;
	long long num4=10000;
	
	cout<<"num1="<<num1<<endl;
	cout<<"num2="<<num2<<endl;
	cout<<"num3="<<num3<<endl;
	cout<<"num4="<<num4<<endl;
	
	system("pause")
	
	return 0;
}
```
## 2.sizeof关键字
- 利用sizeof关键字可以统计数据类型所占内存大小 单位是字节
- **语法：**`sizeof(数据类型/变量)`
```C++
#include<iostream>
using namespace std;

int main()
{
	int num1=10;
	cout<<sizeof(num1)<<endl;
	cout<<sizeof(int)<<endl;
	system("pause");
	
	return 0;
}
```
## 3，实型（浮点数）
用于表示小数 默认情况下输出小数是6位有效数字
浮点数变量分为两种：
1. 单精度float 占用4字节 7位有效数字
2. 双精度double 占用8字节 15/16位有效数字
3. 科学计数法 使用e表示10   3e2表示300 3e-2表示0.03
```C++
#include<iostream>
using namespace std;

int main()
{
	float f1 = 3.14f;
	cout << "f1=" << f1 << endl;

	double d1 = 3.14;
	cout << "d1=" << d1 << endl;

	cout << "sizeof(float)=" << sizeof(float) << endl;
	cout << "sizeof(double)=" << sizeof(double) << endl;

	float f2 = 3e2f;
	cout << "f2=" << f2 << endl;

	float f3=3e-2f;
	cout << "f3=" << f3 << endl;

	system("pause");

	return 0;
}
```
## 4.字符型
用于表示单个字符
**语法：**`char ch='a';`
- 在C/C++中字符型变量只占用1个字节
- 字符型变量本身并不是把字符本身放进内存中储存，而是把对应的ASCII编码放到存储单元
```C++
#include<iostream>
using namespace std;

int main()
{
	char ch = 'a';
	cout << ch << endl;
	cout << sizeof(ch) << endl;
	cout << (int)ch << endl;

	system("pause");

	return 0;
}
```
## 5.转义字符
用于表示一些不能显示出来的ASCII字符
1. `\n`换行 输出后切换到下一行开头  在C++中endl也可以表示换行
2. `\t`制表 输出对齐（相当于按一次Tab）
3. `\r`回车（回到行首）覆盖当前行内容（不换行）
4. `\\`反斜杠 表示路径、反斜杠字符
5. `\"`双引号 字符串中表示单引号
6. `\'`单引号 字符常量中表示单引号
7. `\0`空字符 C风格字符串结束符 
8. `\b`退格（回删一个字符） 删去前一个字符
```C++
#include<iostream>
using namespace std;

int main()
{
	cout << "Hello world\n" << endl;
	cout << "\\" << endl;
	cout << "aaa\thello world" << endl;
	cout << "aa\thello world" << endl;
	cout << "aaaa\thello world" << endl;
	system("pause");
	return 0;
}
```
