> 创建变量或常量时必须指出相应的数字类型，否则无法给变量分配内存

## 1.整型
整型变量表示的是整形类型的数据
short  2个字节
int      4个字节
long   windows占用4字节，linux（32位）占用4字节，（64位）占用8字节
long long  8个字节
```C++
#include<iostream>
using namespace std;

int main()
{
	short num1=10;
	int num2=100;
	long num3=1000;
	long long num4=10000;
	
	cout<<"num1="<<num1<<endl;
	cout<<"num2="<<num2<<endl;
	cout<<"num3="<<num3<<endl;
	cout<<"num4="<<num4<<endl;
	
	system("pause")
	
	return 0;
}
```

