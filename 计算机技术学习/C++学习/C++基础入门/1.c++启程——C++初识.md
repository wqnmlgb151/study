## 1.第一个程序hello world
```C++
#include <iostream>//引入标准输入输出流头文件
using namespace std;//使用标准命名空间

int main()
{
	cout<<"Hello world"<<endl;//输出Hello world并换行 endl是换行符
	
	system("pause");//使运行后的控制窗口暂停
	
	return 0;
}
```
## 2. 注释
>  在代码中加入一些解释和说明，方便自己和其他人阅读代码
>  编译器在编译代码时会忽略注释

**两种格式：**
1. **单行注释：**`//`
2. **多行注释：**`/*...*/`
## 3.变量
**作用：** 给指定的内存空间起名，方便操作这段内存
**语法：**`数据类型 变量名 = 初始值;`
```C++
#include <iostream>
using namespace std;

int main()
{
	int a = 10;
	
	cout<< "a = " << a <<endl//输出 a=10
	
	system("pause");
	return 0;
}
```
