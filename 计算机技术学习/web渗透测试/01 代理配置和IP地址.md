## 3 .ip
ip分为内网ip与外（公）网ip

内网ip：`10.x.x.x`   `172.16.x.x` 到 `172.31.x.x`  `192.168.x.x`

内网与外网：个人电脑ip为内网ip 内网ip可重复，局域网内可重复使用，但无法从外界互联网直接访问

公网ip 地址有限 全球唯一不可重复，可直接互联网访问  
**总结**：内网 IP 是 “家里的小名”（只在内部用，可重名），公网 IP 是 “全球唯一的身份证”（互联网通用，不可重复），两者通过路由器的 NAT 技术 “翻译” 实现互通。
