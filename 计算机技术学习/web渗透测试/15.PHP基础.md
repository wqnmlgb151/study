## 2.PHP基础语法
+ 一般以 <?php开头，以?>结尾	新版php可省略结尾
+ 语句使用分号（;）结尾
+ PHP 文件扩展名通常为 .php（服务器会优先解析其中的 PHP 代码，其余内容按 HTML 输出）

```php
<?php
echo "Hello World!".'<br>';
#echo	打印输出
#'<br>' 换行符，在网页内显示效果，在编译器中只显示前端代码

?>
```

+ 数据类型：
    - String	字符串
    - integer	整形
    - Float	浮点型
    - Boolean	布尔型
    - Array	数组
    - Object	对象
    - NULL	空值
+ 变量声明（美元符开头$，如$value;）
+ 常量声明（define(常量名，常量值)）
+ 调用函数必须使用括号将参数包起来，如：md5("admin");#使用md5加密admin变量
+ php对大小写不敏感
+ 关键字

```php
<?php
  $age = 18;
	echo $age.'<br>';
	$world = 'Hello';
	echo $world;
```

## 4.控制结构
+ if语句	条件控制	可嵌套使用

```php
<?php
  $num = 1;
  if($num>2 and $num==2)//使用and or not与C语言不同
  {
    echo $num;
  }
  else
  {
    echo "num变量过小";
  }
```

+ while、do语句	循环控制

```php
<?php
  $a = 2;
  while($a>=1)
    {
      echo $a.'<br>';
      $a--;
      echo "还剩".$a."杯水";//php中的连接符是.
    }
```

## 5.字符串处理
+ strstr()	字符串的匹配和寻找

```php
<?php
  echo strstr("hello this world","this");//寻找this并返回this及后面的字符串
  echo strstr("hello world","this");//没找到则返回空值，不输出
```

+ str_replace()	替换

```php
<?php
  echo str_replace("CNM","***","田文镜，我CNM");//输出：田文镜,我***
```

+ 单引号与双引号区别：
    - ''	单引号
        * 将内容视为纯文本字符串，不解析变量，变量当普通字符输出
        * 仅支持两种转义字符（\',\\），其他转义字符当普通字符输出
    - ""	双引号
        * 将内容视为解析型字符串，自动解析变量，变量会被替换为实际值
        * 支持更多转义字符

```php
<?php
  $name = "PHP";
  echo '单引号：$name \n';  // 输出：单引号：$name \n（变量和\n未解析）
  echo 'It\'s a cat';       // 输出：It's a cat（仅\'被转义）
  echo "双引号：$name \n";  // 输出：双引号：PHP （换行，\n被解析） 浏览器渲染解析不了\n的换行（仅在纯文本生效） 需要用<br>进行换行
  echo "He said \"Hello\"";  // 输出：He said "Hello"（\"被转义）
```
