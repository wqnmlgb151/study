## 1.用户管理、所有者所属组
-rwxrwxrwx. 1 xiaoming test 90 7月  26 06:32  1.txt

- rwx rwx rwx  3个为1组

&#x20;第一个字母 -:文件类型

&#x9;- 文件

&#x9;d 目录（文件夹）

&#x9;l 链接文档

&#x9;b 可随机存取装置 块设备（比如硬盘、U 盘）

&#x9;c 一次性读取装置 是字符设备（比如键盘、串口）

第一组代表文件所有者对该文件权限

第二组代表所属组对该文件权限

第三组代表其他用户对该文件权限

r:读取

w:写入

x:执行

-:没有权限

1  是链接数  代表文件在系统中有几个分身&#x20;

xiaoming test   分别代表所有者xiaoming  	所属组 test

所有者：该文件被谁创建

所属组：拥有这个文件的用户所在的组

其他用户：不在该组的用户且不拥有该文件

90  代表文件大小	单位字节

后面的是修改日期与文件名字

chmod 更改权限命令

groups 可查看当前用户所在哪组

chgrp 变更文件与目录所属组

chown 变更更文件或目录拥有者和所属组 chown a：b 文件名  即是将该文件的拥有者改为a 所属组改为b&#x20;

chmod 修改读写执行权限 	chmod +r 文件名  所有均有该文件r权限

| `rwx` | 4（r）+2（w）+1（x）=7 | 7 |
| :--- | :--- | :--- |
| `r-x` | 4（r）+0（-）+1（x）=5 | 5 |
| `r--` | 4（r）+0（-）+0（-）=4 | 4 |
| `--x` | 0（-）+0（-）+1（x）=1 | 1 |


例：chmod 754 1.txt  即赋予1.txt rwxr-xr--权限

或：chmod 111 1.txt  即赋予1.txt --x--x--x 权限

也可以通过符号类型进行权限的更改

user：   u

group： g

others：0

all：      a

例：chmod u=rwx 1.txt  即拥有者对1.txt的权限是rwx

或：chmod u=r,g=w,o=x 1.txt

或：chmod u-x 1.txt

## 2.用户管理
useradd 创建新用户

useradd -d /home/sam sam 创建用户sam和sam家目录

useradd -d /home/sam ccc创建用户ccc用sam的家目录

useradd -g aaa sam 创建用户sam并将其加入组aaa   会自动创建其家目录

useradd abc 创建用户abc并创建加入同名组abc 会自动创建其家目录

userdel 删除指定用户&#x20;

userdel -r sam 删除用户sam和家目录sam

userdel -f sam强制删除用户sam即使sam正在登陆 但容易出问题

usermod 修改用户信息

usermod -d /home/xiaoming -g test sam 将用户sam的家目录改为xiaoming，同时将sam分到test组

&#x20;passwd 更改密码  只有root用户可以修改所有用户密码 指定用户只能够修改自己的密码

&#x20;passwd sam 更改sam密码  新密码要符合系统要求强度

groupadd 新建组

groupadd  add  创建分组add

groupdel 删组  直接加组名

groupmod 修改组

gpasswd 组文件管理

gpasswd -a abc xiaoming 将用户abc添加到xiaoming组 但不会脱离原来组&#x20;

gpasswd -d abc xiaoming 将用户abc从xiaoming组中删除

当用户同时在两个组时，创建的文件所属组归优先级高的 一般是同名或先前的组 可通过newgrp  组名 切换组 改变优先级 此时创建文件归该组&#x20;
