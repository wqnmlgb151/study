## 1.ETL数据整合
### ETL
+ 对数据进行抽取、转换和装载的过程，是共享或合并一个及以上的企业应用数据，创建具有更多功能的数据应用过程
+ 抽取	连接到不同数据源
+ 转换	进行数据处理
    - 移动数据
    - 根据规则验证数据
    - 数据内容和数据结构的修改
    - 集成多个数据源的数据
        * 数据整合常用方式：
            + 使用工具软件
            + 编程
    - 根据处理后的数据计算派生值和聚集值
    - 清洗，过滤脏数据
+ 装载	将数据装载到目标数据库和数据文件中

### ETL常用工具
+ DataPipeline
+ Kettle	开源的ETL工具
    - 优势：
        * 开源，无需付费，技术支持强
        * 图形界面，易用性
        * 部署简单，无需安装
        * 强大的基础数据转换和工作流控制
        * 全面的数据访问和支持
        * 要求技能不高，上手容易
+ Oracle Goldengate
+ Informatica

## 2.数据标注和分类
### 数据标注
+ 数据标注是对未处理的初级数据，如图片、视频、文本、语音等数据进行加工处理，并转换为机器可识别信息的过程
+ 数据标注企业主要是根据用户或企业的需求，对图像、文字、声音等对象进行不同方式的标注，从而为人工智能算法提供大量的训练数据以供机械学习使用

### 数据分类
+ 以标记对象为分类基础时，进行分类：
    - 图像数据标注
        * 包括图像标注和视频标注
        * 使用不同的颜色对不同的目标标记物进行轮廓识别，给相应轮廓打上标签，用标签来概括轮廓内容，以便让算法模型能够识别图像中的不同标记物
        * 主要任务：
            + 图像分类
            + 物体检测
            + 描点标准
    - 文字数据标注
        * 根据一定的标准或准则对文字内容进行诸如分词、语义判断、词性标注、文本翻译、主题事件归纳等注释工作
        * 主要任务：
            + 情感标注
            + 实体标注
            + 词性标注
            + 其他文本类标注
    - 语音数据标注
        * 通过算法模型识别转录后的文本内容并与对应的音频进行逻辑关联
        * 主要任务：
            + 语音转写
            + 语音分割
+ 不同标记方式的优缺点：
    - ![](https://cdn.nlark.com/yuque/0/2025/png/59258918/1761640578749-9aacbd16-dc5c-4bf0-9317-4992f877814d.png)
+ 主要应用场景：
    - 出行行业
        * 常见应用：
            + 以矩形框或描点的方式对车辆进行标注
            + 以矩形框或描点标注人体轮廓
            + 采集地址兴趣点
            + 在地图上做出相应地理位置信息标记的POI标记等
    - 安防行业
        * 常见应用：
            + 人脸标注
            + 视频分割
            + 语音采集
            + 行人标注
    - 医疗行业
        * 常见应用：
            + 手术工具标识
            + 处方识别
            + 医疗影像标注
            + 语音标注
    - 金融行业
        * 常见应用：
            + 文字翻译
            + 语义分析
            + 语音转录
            + 图像标注
    - 服务行业
        * 常见应用：
            + 确定内容是否符合标注的内容审核
            + 对具有相同意思的语句进行分类的语义分析
            + 将音频转化为文字的语音转录
            + 查看视频是否符合符合要求的视频审核
