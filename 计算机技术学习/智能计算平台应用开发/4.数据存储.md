## 1.存储系统维护和管理
数据通常使用文件系统或数据库进行存储

### 大数据存储管理
在大数据存储方面也有相应的分布式文件系统和分布式数据库

+ 常见的分布式文件系统：
    - HDFS	分布式、高容错、可先行拓展的文件系统	适用于网站用户行为数据存储、生态系统数据存储、气象数据存储等
        * 将文件进行切片，以多副本（通常是3副本）形式存储
        * 存储量巨增，产生高延迟、不适合低延迟数据访问应用
        * 不适合多用户写入和任意修改文件、HDFS文件是只有一个Writer，且写操作总在文件末尾
        * 不适合大量小文件存储

+ 常见的分布式数据库：
    - Hive		基于Hadoop的数据仓库软件，可查询和管理PB级别的分布式数据
        * 不能在大规模数据集上实现低延迟查询，也不提供实时查询和基于行级的数据更新操作
        * 主要应用于：
            + 海量数据的离线分析（如日志分析、集群状态分析）
            + 大规模的数据挖掘（用户行为分析、兴趣分区、区域展示）
            + 大量数据的汇总（每天/每周用户的点击数、流量统计）

![](https://cdn.nlark.com/yuque/0/2025/png/59258918/1761148250894-eae5d68d-f4d5-469f-b6b5-544594c45134.png)

    - HBase	高可靠性、高性能、面向列、可伸缩的分布式存储系统，提供海量数据存储
        * 存储大表数据，并对大表数据的读、写访问可以达到实时级别
        * zookeeper为HBase集群中各进程提供分布式协作服务

### 数据库存储管理
+ 常用数据库监控工具：
    - SolarWinds Database Performance Analyzer（DPA）	DPA是SolarWinds提供的数据库管理软件
        * 用于SQL查询性能监视、分析和调优，包括用于识别性能瓶颈的工具、响应时间分析和数据库监视
        * ![](https://cdn.nlark.com/yuque/0/2025/png/59258918/1761218022058-14c2d609-0ae6-40c7-8118-50c9b5cd0c34.png)
    - Paessler PRTG	专业数据库监视器
        * 监视IT基础结构中的所有系统、设备、流量和应用程序
    - Idera Dignostic Manager IDM	针对SQL server的强大性能监视、警报和诊断解决方案
        * 可以通过桌面控制台、Web控制台附加组件和移动控制台主动通知健康、性能或可用性问题
        * 提供无代理的实时监视和警报，快速诊断和修复
    - SQL Power Tools	无代理数据哭监控解决方案
        * 监控Informix、MariaDB、MySQL、Oracle、SQL server和Sybase数据库服务器，通过数据包嗅探来分析服务器的性能
        * 结合轻量级SQL服务器监控和网络安全功能

## 2.存储系统优化
### 负载平衡
+ 静态负载平衡
+ 动态负载平衡

### 数据存储的安全性
+ 存储策略	在存储系统中提高数据安全性的一个重要方法就是对数据进行冗余备份存储
    - 常用的冗余备份机制：
        * 完整文件副本
        * 文件分块副本
        * 独立冗余磁盘阵列（RAID）

### 数据组织结构及复制策略
+ 目前主流的数据结构
    - P2P数据组织方式（分布式存储）
        * Facebook的Cassandra
        * Amazon公司的Dynamo
    - 元数据服务器数据组织方式
        * DFS
        * 开源的HDFS
        * Ceph
+ 复制策略
    - 静态复制
    - 动态复制
